In order to input key definitions, you must first choose a key from the visualized keyboard and then input the new keydefinition to the textbox at bottom.

Use the keynames shown on the visualized keyboard, or alternatively, EVDEV-names.

For simultaneous keystrokes, separate keys with ':'. For separate strokes, use '|'. You can chain these if you want - examples: k:i:s:s:a, k:i:s:s:a|k|i|s|s|a, a:b:c|c:b:a. For quickly defining a string with no special symbols to a key, use write("string"). To repeat something, use repeat("string/char/key",count).

To define a key that goes through profiles, use chProf("Profile-1, Profile-2 ...") or else chProf("1,2,3"). You may use either profile names or IDs. To define a mode change button, use chMod("Profile-1/1",booleanfortoggle).

In order to submit a change locally, click Submit. To post it to the server, click Post.
